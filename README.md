# asciicast2png

[asciicast2gif](https://github.com/asciinema/asciicast2gif) patches to write the final frame of the gif as png

## Usage

```sh
git clone https://github.com/hyperupcall/asciicast2png
cd asciicast2png
./bake init
./bake build

# To run the example
./bake prerun
./bake run
```
